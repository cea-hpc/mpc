Mode Hooks library

group MPC Library mode hooks

Libmode allows MPC to run alongside another MPI instance to do so simply compile mpc with .

/installmpc &#8212;lib-mode. Note that a TCP rail is required to bootstrap the comm ring to enable on demand connections on high speed networks.

These function are the weak implementation to be overridden by actual ones in the target implementation.
