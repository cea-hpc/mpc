Current dependencies for this project are :

 - BUILD \
         + gcc             (tested version 4.7.3)
         + g++             (tested version 4.7.3)
         + cmake           (tested version 2.8.12)
         + make            (tested version gmake-3.82)
         + OpenPA          (tested version 1.0.2)
         + Hwloc           (tested version 1.6.1) [OPTIONAL]
 - RUN \
         + Hwloc           (tested version 1.6.1) [OPTIONAL]

Tested systems are :

 - Gentoo x86_64 (2013)
 - Centos 6.2
 - Centos 5.3

Tested compilers are :

 - gcc/g++ 4.7.3
 - gcc/g++ 4.4.5
 - icc/icpc
